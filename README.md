
# BIS202 Object Detection

## Project Overview

This project implements a cloud-based object detection system using Amazon Web Services (AWS) and the YOLOv5 deep learning model.

## AWS Services Used

- Amazon EC2
- Amazon S3
- AWS Lambda
- Amazon API Gateway
- AWS Systems Manager (SSM)
- IAM

## Features

- Upload images to Amazon S3
- Process images using YOLOv5 on EC2
- Trigger processing through AWS Lambda
- Upload processed images to Amazon S3
